O time

Quem faz o BMAD Studio.

O portal nasceu de uma pergunta simples: e se o time de produto pudesse usar os modelos do Copilot — que a empresa já paga — com método, contexto e as ferramentas internas, sem depender de chave de API nem de fila de infraestrutura?

O que a gente acredita

  • IA com método. Modelo bom sem processo vira brainstorm infinito. O BMAD Method dá o trilho: pesquisa, brief, PRD, épicos, código, revisão.
  • Local e auditável. Seus dados no seu disco, código aberto para o time ler, e nenhuma chamada além das que o VS Code já faz.
  • A rede corporativa é requisito, não obstáculo. Proxy, CA, registry e RACF fazem parte do produto — por isso o Diagnóstico e o bootstrap existem.
