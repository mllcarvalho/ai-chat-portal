      4
      
    Aprovar o portal no VS Code
    
      pendente
    
    
    
    
      Ao iniciar a primeira conversa, o portal pede uma aprovação
      dentro do VS Code. Deixe o VS Code aberto, olhe a janela dele
      e aceite a solicitação que aparecer — sem essa aprovação a
      conversa não anda.
    
    

    
    
      Não apareceu nada? Clique na janela do VS Code para trazê-la para frente —
      o aviso pode estar esperando lá. Isso acontece só na primeira vez.
