BMAD Studio · documentação

Chat com os modelos do Copilot, feito para o trabalho de produto.

O BMAD Studio é um portal de chat que roda na sua máquina e usa os modelos do GitHub Copilot pelo login que você já tem no VS Code. Sem chave de API, sem custo extra — com projetos, arquivos, skills, agentes, bases de conhecimento, o BMAD Method e integrações MCP.

como funciona — tudo em 127.0.0.1
┌──────────────┐    HTTP + SSE    ┌───────────────────────────┐
│  Navegador   │ ◄──────────────► │  Extensão VS Code (proxy)  │ ──►  vscode.lm (Copilot)
│  (React UI)  │    127.0.0.1     │  servidor local com token  │ ──►  MCPs do VS Code
└──────────────┘                  └───────────────────────────┘
# nada sai da sua máquina além das chamadas que o próprio VS Code já faz

O que dá para fazer

chat

Conversas com modos

Ask (pergunta e resposta), Plan (gera plano, só leitura) e Agent (usa ferramentas e mexe nos arquivos).

 projetos

Projetos e arquivos

Cada projeto tem uma pasta no disco. O assistente lê e escreve nela; você acompanha tudo pelo painel de arquivos.

 método

BMAD Method

Personas e workflows do BMAD v6 integrados: da pesquisa ao PRD, épicos, arquitetura e revisão.

 integrações

MCPs

ConsumerLab (AWS) e IUClick (ServiceNow) com setup guiado, além dos MCPs que você já usa no VS Code.

 reuso

Skills e agentes

Instruções reutilizáveis, comandos slash e presets de agente — exportáveis em .zip para compartilhar com o time.

 contexto

Bases de conhecimento

Documentos e URLs que entram como contexto das conversas. Importa SharePoint, páginas web e arquivos locais.

Princípios

  • Local primeiro. A extensão sobe um servidor em 127.0.0.1 protegido por token; o navegador conversa só com ela. Seus dados ficam no seu disco.
  • Sem custo novo. As chamadas de modelo passam pela API oficial do VS Code (vscode.lm) — a mesma licença Copilot, a mesma cota.
  • Feito para a rede corporativa. Proxy com RACF, CA interna, registry Artifactory e a tela de Diagnóstico que confere tudo e diz o que falta.
  • Time de produto no centro. As ações BMAD ficam a um clique na conversa: brainstorm, pesquisa, brief, PRD, épicos — cada uma já com a persona certa.

Requisitos em uma linha

VS Code com conta GitHub logada (Copilot habilitado) + Node.js 18+. Windows ou macOS. O portal vive enquanto houver uma janela do VS Code aberta — a extensão é o servidor.

Primeira vez aqui? Siga o passo a passo de instalação — ele verifica os programas da sua máquina e configura a rede corporativa por você.
